Tensorspline#

class splineops.interpolate.tensorspline.TensorSpline(data, coordinates, bases, modes)[source]#

Bases: object

Represents a tensor spline model for multidimensional data interpolation using spline bases and extension modes.

Parameters (__init__())#

  • data (numpy.typing.NDArray) – The data array to fit the tensor spline model.

  • coordinates (Union[numpy.typing.NDArray, Sequence[numpy.typing.NDArray]]) – The coordinates corresponding to each dimension of the data.

  • bases (TSplineBases, None) – Spline bases for each dimension.

  • modes (TExtensionModes, None) – Extension modes for each dimension.
